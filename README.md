# Calculadora ANS

## Requisitos
- Django
- numpy          
- pandas   
- scipy          
- sympy  

## Recomendaciones
- Crear entorno virtual 
```
python3 -m venv envCalculadora
```
- Activar entorno virtual
* GNU/Linux
```
source venv envCalculadora/bin/activate 
```
* Windows
```
envCalculadora\Scripts\activate.bat
```

## Uso
```
python manage.py runserver
```
